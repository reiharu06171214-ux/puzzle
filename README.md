# 15パズルゲーム

Java で作成したシンプルな 15 パズルゲームです。  
マウス操作でパネルを動かして、1〜15 を並べて完成を目指します。

## 🎮 遊び方

- ゲーム画面をクリックすると開始します。
- 数字のパネルをクリックすると、空いているマスに移動します。
- 1〜15 が順番に並べばクリアです。

## 🧩 機能

- 15 個のパネル画像を読み込んで表示
- マウスクリックによるパネル移動
- クリア判定機能
- ランダムシャッフル
